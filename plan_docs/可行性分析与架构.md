思维导图核心内容总结
 
一、核心技术与框架
 
- 知识图谱、记忆库
- 工作流框架：comfyui、n8n pipeline框架
- 生成与处理技术：图像生成框架、3D反推、OCR/图像识别与处理、多模态结合
- 大模型相关：llm for style、风格文本for llm、长上下文llm、RAG+知识图谱（知识获取）
- 其他技术：强化学习方向、推荐算法
 
二、数据相关
 
- 训练数据集：clothes（kaggle/huggingface来源）穿搭图片训练集、多模态数据集
- 数据标注：Label Studio 数据标注服务商/自动打标
- 潮流资源：期刊/杂志动态潮流资源
 
三、平台与资源
 
（一）应用平台
 
- 电商平台：淘宝、京东、亚马逊
- 社交平台：小红书，抖音
- 云服务器平台、模型/数据集微调平台
 
（二）核心资源
 
- 云算力、图像生成云接口、Api接口与资费
- 专家/顾问、种子用户
- 反馈工具、展览
 
四、应用场景
 
- 虚拟试穿
- 风格化对应
 
五、关键考量
 
- 技术可行性、可行性与方案选型
- 爬虫策略（社交平台爬虫限制）
- 商用模型和框架
- 隐私保障
